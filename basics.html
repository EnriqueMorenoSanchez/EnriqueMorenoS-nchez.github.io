Intro to CSS

Id and Class

For adding an Id to a paragraph, for example, you need to add inside
the "<"p">", "Id", then "=" and give a name after that between "_". Afer that
you just need to create inside "<"head">" "<"style">" and then add the following thing:

#name{
}

Same happens with Class, instead of using Id, you select it as "class" and then instead of using "#", you use "."

You can find examples of both methods in the example of unorderlist, where all the white-blue lines are inside a class,
while the red-blue one has an id.

Select by tag name

It is similar to what you have done in the previous step. However, you won't use any Id or Class.
You are just going to type the exact thing you want to select. For example, a paragraph: You create
inside "<"style">":

p{
}

You can see these in the titles of this webpage.

In addition, you add properties to what you have selected, like adding colors,etc.
For adding colors you write inside "color:" and then "rgb()" or just the name of
the color
