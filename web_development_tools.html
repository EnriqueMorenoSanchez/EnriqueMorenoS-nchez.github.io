Web Development Tools

Browser Debug Tool

Every modern web browser includes a powerful suite of developer tools. These tools do a range of things, from inspecting currently-loaded HTML, CSS and JavaScript to showing which assets the page has requested and how long they took to load; and you can make changes in the code and see how the creators of the website designed the webpage.

You can open it through:

  • Keyboard:
    • Ctrl + Shift + I
    • f12 (in Windows OS)
    • ⌘ + ⌥ + I (Mac OS).
  • Menu bar:
    • Firefox: Menu ➤ Web Developer ➤ Toggle Tools, or Tools ➤ Web Developer ➤ Toggle Tools
    • Chrome: More tools ➤ Developer toolsz
    • Safari: Develop ➤ Show Web Inspector. If you can't see the Develop menu, go to Safari ➤ Preferences ➤ Advanced, and check the Show Develop menu in menu bar checkbox.
    • Opera: Developer ➤ Developer tools
  • Context Menu:
    • Press-and-hold/right-click an item on a webpage (Ctrl-click on the Mac), and choose Inspect Element from the context menu that appears.

In the top left corner, there is an icon that lets you select an element in the page to inspect it. You can do it by using Ctrl + Shift + C too.
