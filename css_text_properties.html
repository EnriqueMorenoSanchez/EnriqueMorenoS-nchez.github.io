CSS Text Properties

Generic Font Family

There are different types: "serif", "sans-serif", "cursive", "fantasy", "monospace".

For adding one, type the following inside the item style in the style tag:
font-family: generic_font_name;

Example: Hello World

Specific Font Family

There are different types: "times", "courier", "arial", etc.

For adding one, type the following inside the item style in the style tag:
font-family: "specific_font_name", generic_font_name;

Example: Hello World

font-size

font-size allows you to change the size of the text, there are different ways, 2 of them are "em" and "px".

  • em (include inside the item you want to style in the style tag):
    font-size: Amount_em;
    Example: Hello world
  • px (include inside the item you want to style in the style tag):
    font-size:Amount_px;
    Example: Hello world

font-style

It lets you include text with different styles, like italic. (font-style: italic;)

Example: Hello world
