Target

For sending someone to another webpage there are 2 options:

  1. In a new window of the browser (target="_blank" href="url" or href="file"). Example: Link to Wikipedia.
  2. In the same windows of the browser (href="url" or href="file" ). Example: Link to IES Gabriel Alonso de Herrera.

Absolute and Relative

The both are used for moving an image or text to a specific place in the webpage.

  • Absolute: Used when you want to establish the item as the first thing of the page, without being affected by other elements(text,etc) when it is placed.
  • Relative: The item is relative to other elements of the page, so the place it will be established depends on other items.

You can see their examples and also more information in CSS Layout.
